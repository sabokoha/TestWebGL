起動時の設定変更と終了処理の説明

基本編の最後です。今までので一通り、音楽と絵はついたと思います。
そこで終了処理について最後やっていきましょう。
minisina.txtの一番最後にed,1、終わりというのが書いてあるかなと思います。
こちらが終了処理で、今の所タイトルに戻る機能しか実装されていません。
今後、幾つかのシナリオをやったり、ミニゲームなどの実装をした際などに使っているタグなので、そういうものとして覚えておいてください。
 ed,1
終わり
さて、今までシナリオモード内での動作はやりましたが、起動時の画面等は変えませんでした。
ですが、実際はその辺も必要になるので最後に起動時の絵やBGMの設定を行います。
ConfJS.txtを開いてください。
"BgmDat":"",の空間の部分に音楽データの名前を入れます。
ttmusic1,mp3という名前を入れればOKです。
"BgmDat":"ttmusic1,mp3",となります。
"TitleBg":"DOUSAP,png",となっていますが、こちらも先ほど同じように、絵のデータを入れていきます。
"TitleBg":"PLUSG,png",という絵のデータを入れてみました。
後は実際にこれを動かしてみて確認してみてください
 {
"TitleDT":[
{
"TitleBg":"DOUSAP,png",
"BgmDat":"ttmusic1,mp3",
"LoadS":"true,LOAD,640,360",
"ConfigS":"true,Config,640,360",
"GMEnd":"true,ゲーム終了,460,100"
}
]
}
これで、起動して音楽と絵が出てシナリオも出来るようになりました。
今までの物で一通り基本設定は終了です。
基本設定編を一通りまとめたものをトップページの別ファイルに纏めておいたので、そちらで確認してみましょう。
